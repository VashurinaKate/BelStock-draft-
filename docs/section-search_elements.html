11 #Search_Elements Search_Elements

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

Элементы поиска и значок просмотра корзины. Важно не забыть добавить в html в head <link rel="stylesheet" href="font-awesome-4.7.0/css/font-awesome.min.css"> , чтобы появился значок корзины. В данном случае в тег button добавляется <i class="fa fa-shopping-cart" aria-hidden="true"></i>

Examples
Default styling
.search--main
лупа с заливкой
.search--transp
лупа на прозрачном фоне
.search--cart
просмотр корзины
Markup 
<link rel="stylesheet" href="font-awesome-4.7.0/css/font-awesome.min.css">
<div class="search [modifier class]"><i class="fa fa-search"></i></div>
<div class="search search--transp"><i class="fa fa-search"></i></div>
<div class="search search--main"><i class="fa fa-search"></i></div>
<div class="search search--cart"><i class="fa fa-shopping-cart"></i></div>
