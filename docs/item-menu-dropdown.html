8.1 #Menu.Dropdown Dropdown

Toggle example guides Toggle HTML markup

Выпадающее меню при наведении. Чтобы добавить выпадающее меню, необходимо добавить к элементу li класс <li class="drop-link">

Example
  • Продукты

    • IP-оборудование
    • HD-TVI - оборудование
    • Домофония
